# Pinewood Derby AI-Assisted Design Assets

This repository contains the code, CAD models, vector designs, and physical template assets created to support the building of two custom-engineered [Pinewood Derby](https://en.wikipedia.org/wiki/Pinewood_derby) cars in early 2026. The cars were designed and built by [@dt1900](https://github.com/dt1900) his children with the assistance of [Google Gemini](https://gemini.google.com) as a workshop tool.

The complete story, building methodology, and race results are documented in the accompanying blog post: [https://medium.com/@_DANTHOMPSON/pinewood-ai-a9eb03ba1b30](https://medium.com/@_DANTHOMPSON/pinewood-ai-a9eb03ba1b30).

## Repository Directory Structure

This project is organized into two primary folders, each dedicated to a car design, alongside configuration files.

### 🐟 1. The Goldfish Car ("Fish 1")
A goldfish-themed car featuring a classic wedge shape wrapped in a procedurally generated "Goldfish Cracker" livery and topped with a swap-out 3D-printed acrylic fishbowl filled with clear epoxy.

- `/goldfish/`
  - [goldfish.py](./goldfish/goldfish.py): A [Python](https://www.python.org) script utilizing the [Pillow](https://python-pillow.org) image processing library. It uses mathematical rejection sampling to generate a high-DPI background pattern of scattered goldfish crackers that fades out from left to right.
  - [mclaren_goldfish_livery_600dpi.png](./goldfish/mclaren_goldfish_livery_600dpi.png): The output gradient pattern generated by the script, ready to be imported into vector design software.
- `/goldfish/livery/`
  - [livery.svg](./goldfish/livery/livery.svg): The master vector layout created in [Inkscape](https://inkscape.org), combining the Python-generated goldfish cracker gradient pattern with official [BSA](https://www.scouting.org) and custom F1-style logos.
  - [mock.svg](./goldfish/livery/mock.svg): An archive layout containing early draft designs.
  - [livery-600.png](./goldfish/livery/livery-600.png): High-resolution sticker print export.
  - [Scan 23.jpeg](./goldfish/livery/Scan%2023.jpeg), [Scan 24.jpeg](./goldfish/livery/Scan%2024.jpeg): High-resolution scans of physical paper print tests.
  - [img/](./goldfish/livery/img/): Subfolder containing source branding logos (`F1-logo.png`, `BSA` style assets) and the final livery preview image (`image.png`).
- `/goldfish/stl/`
  - [fishbowl.stl](./goldfish/stl/fishbowl.stl): 3D-printable model of the clear acrylic fishbowl, designed with a bottom peg for modular mounting.

### 🐹 2. The Hamster Wheel Car ("The Sakura Spinner")
A Pinewood Derby car featuring a central 3D-printed hamster wheel that spins freely as the car rolls. Decorated in a cherry blossom (Sakura) theme.

- `/hamster/`
  - [cage3.scad](./hamster/cage3.scad): A parametric [OpenSCAD](https://openscad.org) script used to compile and render the spinning hamster cage. It features a custom double-helix twist lattice structure designed to withstand high-velocity runs.
  - [cage.stl](./hamster/cage.stl): The 3D model of the hamster wheel cage. This file is version controlled in Git, with the design history (v1 baseline, v2 twist lattice, v3 final 18-pillar layout) stored directly in its commit timeline.
- `/hamster/templates/`
  - [top.svg](./hamster/templates/top.svg) & [side.svg](./hamster/templates/side.svg): Vector cutting templates calculated with AI to guide the physical wood cuts. These ensure the structural integrity of the side supports is maintained around the central wheel well.
